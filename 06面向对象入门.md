### 面向对象

将程序模块化，关注于**复用性**和**灵活性**

#### 类和对象

用**属性**和**方法**来描述每一个对象的特征

属性是静态特征，方法是动态特征

对象是用来描述客观存在的实体，该实体是由一组属性和方法组成

**类是产生对象的模板，所有的对象都是通过类来创建的**

- 类是对象的抽象化描述，这些对象具有相同的特征和动作
- 对象是类的具体实现

#### 定义类

```java
public class 类名{
    //定义属性，属性名符合驼峰命名
    public 数据类型 属性名;
    //定义方法，符合驼峰命名
    public 返回值类型 方法名(参数列表){
        //方法体
    }
}
```

#### 构造函数、构造方法、构造器

##### **构造方法**

是一种特殊的方法，用来**创建对象**

- **方法名必须与类名一致**
- **不需要定义返回值类型**

无参构造：每个类自带一个无参构造，但手动给任一个有参构造，就不给默认无参构造

##### this关键字

this用来指代**当前类的实例化对象**，通过this可以调用当前类的属性和方法

this调用构造方法：`this(参数列表)`，**只能在构造函数中使用**，而且**只能出现构造函数第一行**
this调用普通方法：`this.方法名(参数列表)`，普通函数和构造函数都能用
