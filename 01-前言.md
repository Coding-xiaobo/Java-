## 前言

### Java语言特点

- 简单：相比C++有垃圾回收机制，不需要考虑对象的回收
- 跨平台性：支持Java虚拟机（JVM，屏蔽不同操作系统的底层），就都可以运行Java
- 稳定
- 需求量大

### Java运行机制

1. **编写**Java程序
2. **编译**Java成字节码文件
3. JVM读取字节码文件，**运行**程序

```java
//Java源程序通过编译器编译成字节码文件，字节码文件运行到虚拟机里，进而在不同操作系统中执行
```

### Java三大体系

- JavaSE：Java核心类库和常用的组件
- JavaME：专门用于移动开发
- JavaEE：JavaWeb组件

### 开发

- 编译

```java
javac HelloWorld.java
```

- 运行

```java
java HelloWorld
```

### 代码规范

- 强制性代码规范
  1. Java程序的**文件名和类名必须一致**
  2. main是程序的入口，只能有一个main
  3. 类是组织java代码结构的，类中的方法是执行具体业务的
- 非强制性代码规范
  1. 一行只能写一条语句
  2. 注意代码缩进
